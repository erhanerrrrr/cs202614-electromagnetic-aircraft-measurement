# 文献目录说明

本目录作为文献资料入口。当前项目的结构化文献矩阵仍保留在 `docs/` 中，便于报告脚本和阶段说明继续引用。

## 结构化文献文件

| 文件 | 说明 |
|---|---|
| `docs/literature_matrix.md` | 权威文献矩阵，按测量规范、近远场变换、压缩采样、RF 指纹等方向整理。 |
| `docs/literature_screening_and_strategy.md` | 文献筛查原则与可迁移技术策略。 |
| `docs/literature_to_algorithm_traceability.md` | 文献结论到本项目算法/文件/待补证据的映射。 |
| `docs/future_engineering_plan.md` | 本轮结合导师 DeepSeek 记录和新增推荐文献形成的未来方案。 |

## 新增重点文献方向

- 电磁场自由度、NDF 与非冗余采样。
- 球面近场到远场变换的标准工程链路。
- 受限空域压缩测量和多频联合稀疏/低秩建模。
- 面向半球面 2π 测量的非均匀、非冗余、主动采样策略。
