## 2. 总体技术路线

本方案采用“由简到繁、先标准源校准、再复杂载体扩展”的路线，避免直接在复杂模型上验证导致问题不可定位。

```text
赛题要求拆解
  -> 文献调研与方法选型
  -> 2π半球面/半柱面测量布局设计
  -> CST Level 1标准源仿真
  -> CST近场/远场数据导出与格式校验
  -> 等效源反演与远场重建
  -> 测点删减和优化
  -> CST Level 2多源多状态仿真
  -> 空间-频谱-极化特征提取
  -> 不同载体/状态识别
  -> 报告、PPT、视频、代码和CST工程提交
```

### 2.1 测试系统路线

1. 测量面选择：当前主方案采用2π上半球测量面，半柱面作为备选扩展。
2. 被测包络：覆盖不小于12m x 10m x 8m的空间范围，坐标取x方向±6m、y方向±5m、z方向0-8m。
3. 测点配置：半球半径约13m，初始9个俯仰角层 x 18个方位角 = 162个空间测点。
4. 极化通道：每点至少保留两个正交极化响应；CST中优先导出Ex/Ey/Ez复数场，再投影到theta/phi或H/V。
5. 数据接口：所有CST导出统一转换为CSV，字段必须包含`sample_id`、`sensor_id`、坐标、频率、极化、复数实部/虚部、源状态和工程名。

### 2.2 仿真路线

| 层级 | 目的 | 模型 | 通过标准 |
|---|---|---|---|
| Level 1 标准源 | 验证坐标、相位、极化和重建链路 | 短偶极子、半波振子、小环/贴片 | CST远场与Python重建远场相关系数高，主瓣误差小 |
| Level 2 多源多状态 | 构造识别数据集 | 2-4个辐射源，多频点，多工作状态，多极化 | 形成可训练的空间-频谱-极化样本，识别精度达到85%以上 |
| Level 3 简化航空载体 | 引入载体结构影响 | 机身、机翼、尾翼、金属/介质结构 | 展示遮挡、散射、副瓣和安装位置影响 |
| Level 4 扰动鲁棒性 | 证明工程适用性 | 噪声、缺测、姿态扰动、频点偏移 | 给出性能随扰动变化的曲线和阈值 |

### 2.3 算法路线

近场测量向远场推算采用等效源思想。将被测载体包围在一个等效源面内，用近场测量值反演等效电流/磁流，再由等效源外推远场。

核心模型写成：

```text
y = G j + n
```

其中`y`为传感器测得的复数近场，`G`为等效源到测点的传播矩阵，`j`为等效源系数，`n`为噪声。基本反演采用Tikhonov正则：

```text
j_hat = argmin ||W(Gj - y)||_2^2 + lambda ||Lj||_2^2
```

远场外推为：

```text
f_hat(theta, phi, freq) = H(theta, phi, freq) j_hat
```

评价指标：

| 指标 | 含义 | 用途 |
|---|---|---|
| NMSE | 归一化均方误差 | 衡量整体方向图误差 |
| Correlation | 方向图相关系数 | 衡量重建形状一致性 |
| Main-lobe error | 主瓣方向误差 | 衡量主要辐射方向是否正确 |
| Peak gain error | 峰值误差 | 衡量强度标定可靠性 |
| Sensor count | 测点数量 | 对应客观分“测点尽量少” |
| Accuracy/F1 | 识别准确率和类别均衡性能 | 对应85%辨识要求 |

## 6. 关键技术细节

### 6.1 半球面测量布局

- 测量面：2π上半球。
- 半径：约13m，用于包络12m x 10m x 8m目标并保留余量。
- 初始测点：162个空间点。
- 测量通道：双极化时等效324通道；三分量场时可保留Ex/Ey/Ez。
- 报告中必须说明坐标原点、x/y/z方向、theta/phi定义和频率单位。

### 6.2 CST导出字段

每条近场记录至少包含：

```text
sample_id, sensor_id, x_m, y_m, z_m,
theta_deg, phi_deg, frequency_hz,
polarization, e_real, e_imag,
source_config, carrier_model, working_state, cst_project
```

每条远场真值记录至少包含：

```text
sample_id, theta_deg, phi_deg, frequency_hz,
polarization, e_real, e_imag, gain_db
```

禁止只保存幅度图而不保存数值表。若CST输出幅度/相位，必须转换成实部/虚部再进入重建算法。

### 6.3 重建算法细节

1. 先用Level 1标准源验证坐标和极化，不直接上复杂载体。
2. 等效源面放在被测包络外、测量面内，避免源面与测量面过近导致病态。
3. 正则化参数通过扫描选择，报告中给出`lambda`对NMSE和主瓣误差的影响。
4. 测点优化采用“全测点基准 -> 随机删减 -> 几何均匀删减 -> 基于互相关/信息量的优化删减”的顺序比较。
5. 输出不只给漂亮方向图，还必须给指标表和误差图。

### 6.4 特征辨识细节

候选特征包括：

- 空间方向图峰值、主瓣方向、半功率波束宽度、副瓣水平。
- 多频点方向图变化率、频谱峰值、带内能量分布。
- 极化比、交叉极化水平、theta/phi通道相关性。
- 等效源稀疏系数、源面能量质心、源面分布稳定性。

训练和验证要求：

1. 样本划分必须按`sample_id`划分，避免同一仿真状态的不同频点同时进入训练和测试导致数据泄漏。
2. 指标至少包含accuracy和F1；类别不均衡时优先看macro-F1。
3. 赛题要求85%是最低线，正式报告目标应写成“稳定超过85%并给出误差来源”。

### 6.5 报告和PPT证据链

每个结论必须对应一个证据文件：

| 结论类型 | 必要证据 |
|---|---|
| 测量面覆盖 | 布局图、测点表、12m x 10m x 8m包络说明 |
| 远场重建有效 | CST远场真值、Python重建结果、误差指标 |
| 测点减少可行 | 不同测点比例下的误差曲线 |
| 识别达到85%以上 | 真实或最终仿真数据上的accuracy/F1/混淆矩阵 |
| 工程可复现 | 代码、requirements、复现命令、数据字典 |

|---|---|---|
| CST导出缺少相位 | 等效源重建精度下降 | 优先导出复数场；若只能幅相导出，统一转换；若只有幅度，启用无相位重建备用说明 |
| 坐标/极化定义混乱 | 重建方向图错位 | Level 1先做x/y/z标准源 sanity check，报告固定坐标图 |
| Level 2样本量不足 | 识别指标不稳定 | 先做四类pilot，确认流程后批量扩展到48个样本 |
| 测点减少后主瓣偏移 | 客观分受影响 | 以75%测点作为稳健方案，50%作为挑战方案，25%只作为边界对照 |
| 报告结论只有demo证据 | 不能证明赛题完成 | 所有正式结论替换为真实CST或最终仿真数据，demo只放附录说明管线可运行 |

## 10. 一句话版本

本队方案以华尔涵负责总体路线与重建算法、张昊负责CST真实仿真数据、李伊然负责文献数据规范与识别展示为核心分工，采用2π半球面宽带传感布局，通过CST标准源和多源载体仿真获取近场/远场数据，再用等效源重建、测点优化和空间-频谱-极化特征辨识完成赛题要求的三维场域重建和不低于85%的状态识别。
