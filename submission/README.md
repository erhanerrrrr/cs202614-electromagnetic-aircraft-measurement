# submission 目录说明

本目录是当前提交草稿/交付结构，按比赛材料组织为报告、PPT、视频、代码、CST、数据和附录。

## 子目录

| 子目录 | 说明 |
|---|---|
| `00_admin/` | 报名与提交信息模板，需要队伍人工填写。 |
| `01_report/` | 正式报告 DOCX/PDF 与报告状态说明。 |
| `02_presentation/` | 答辩 PPTX/PDF、PPT 大纲和素材包。 |
| `03_video/` | 演示视频与视频脚本/素材。 |
| `04_code/` | 提交包内的代码快照、依赖与复现说明。 |
| `05_cst/` | CST workpack、宏模板、工程索引、运行日志和结构对照。 |
| `06_data/` | 数据、指标、处理输出和审计材料。本目录体量较大，GitHub 默认排除部分可再生成数据。 |
| `07_appendix/` | 文献、进展、阶段说明和补充材料。 |

正式报送前仍需人工检查学校、队名、申报人、联系方式、报名表和视频讲解口径。
